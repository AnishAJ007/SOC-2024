# Data Structures and Algorithms Solutions

This repository contains various solutions to common data structures and algorithms problems implemented in C++. The solutions cover a range of topics such as linked lists, arrays, recursion, and more. Below is a brief overview of each implemented solution:

## Solutions Overview

1. **Data Type Size**
   - **Description**: A function to determine the size of various data types such as `Character`, `Integer`, `Float`, `Long`, and `Double`.
   
2. **Second Largest Element in Array**
   - **Description**: A function that finds the second largest element in an array.
   
3. **Find Peak Element**
   - **Description**: A function to find a peak element in an array where a peak element is an element that is greater than its neighbors.
   
4. **Delete Middle Node in Linked List**
   - **Description**: A function to delete the middle node of a singly-linked list.
   
5. **Detect Cycle in Linked List**
   - **Description**: A function to detect a cycle in a singly-linked list.
   
6. **Check Palindrome in Linked List**
   - **Description**: A function to check if a singly-linked list is a palindrome.
   
7. **Reverse Linked List**
   - **Description**: A function to reverse a singly-linked list using recursion.
   
8. **Sort Linked List**
   - **Description**: A function to sort a singly-linked list using the merge sort algorithm.
